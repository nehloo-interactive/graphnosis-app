        🧠 Graphnosis Neural Network (.GNN)
        
                        
        🔮 A little neural net learns the shape of your graph and guesses connections you might've missed — hunches, not facts.
        
                        
        
                          
        🔒 .gai your real memories untouched
        
                          
        🧠 .gnn predicted edges — guesses lives here
        
                        
        
                        
          
                          
        • 🔮Predictions only — never your notes, never the canonical graph.
          • 
                          
        • 🔐Its own encrypted file beside your cortex: neural-network.gnn.
          • 
                          
        • ⏸️Off by default — a full snapshot is saved before it's ever enabled.
          • 
                          
        • 🗑️Delete the file → back to the exact pre-GNN state, byte-for-byte.
          • 
                        
        
                        
        
                      
        

                      
        
                        
        ✨ Graphnosis Local Layer (.GLL)
        
                        
        💡 Your on-device LLM quietly proposes links between related memories. You decide what sticks.
        
                        
        
                          
        🔒 .gai your real memories untouched
        
                          
        💡 .gll LLM's proposed links review below
        
                        
        
                        
          
                          
        • ✅Accept → it becomes a real edge.  ❌ Reject → gone for good.
          • 
                          
        • 🔐Encrypted overlay (local-layer.gll) — the LLM can't touch your attested memory.
          • 
                          
        • 🤖Needs Predict edges autonomously under Local LLM.
          • 
                          
        • 🗑️Delete the file → back to the pre-LLM state.
          • 
                        
        
                        
                        
        
                          
        .GLL autonomous edge prediction is a Pro feature
        
                          
        The background loop that asks your local LLM to propose
                          new connections between memories runs only with a Graphnosis Pro
                          subscription. The .gll overlay stays encrypted on your

                    Ingest

            many texts it groups into each embedding model call. Sensible defaults
            cover most users; tune if ingest feels slow or recall feels imprecise.
            Changes apply to the next file you ingest. To apply the

            2 = default, 3–4 = faster bulk ingest on capable machines. Change takes
            effect immediately without restarting.

            is taken first so the old chunks/vectors are recoverable.

            Cache content set to off) are skipped — they'd need to be
            re-ingested from the source file or URL.
